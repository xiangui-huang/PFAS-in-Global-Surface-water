This repository contains the source code developed for the study “Global Divergence of Per- and Polyfluoroalkyl Substances in Lotic and Lentic Surface Waters.”

Contents

This repository includes:

Python code for resampling nighttime light intensity from raster datasets.
Note: The nighttime light raster files must be downloaded separately using the link provided in the file URL for downloading the VNL raster data.txt.

JavaScript code for land-use and land-cover (LULC) extraction using Google Earth Engine (GEE).
Users must register for a Google Earth Engine account before running the script.

R code for Bayesian Network structure learning and analysis.
